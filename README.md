Study materials, walkthroughs, and practice projects for the **Microsoft AI-900: Azure AI Fundamentals** certification.

AI-900 covers core AI and machine learning concepts, and the Microsoft Azure AI services — including Azure AI Vision, Language, Speech, Document Intelligence, and Generative AI. It sits alongside AZ-900 as a fundamentals-level cert and is a natural next step for anyone on the Power Platform path, given how deeply AI Builder and Copilot Studio integrate with these services.

## Who this is for

- Anyone preparing for the AI-900 exam
- Power Platform makers wanting to understand AI Builder and Copilot Studio more deeply
- Developers exploring Azure AI services for Power Platform integration
- Career switchers building a broader Microsoft cloud credential portfolio

## What this repo will cover

- Fundamental AI and machine learning concepts
- Azure AI services: Vision, Language, Speech, Document Intelligence
- Generative AI and Azure OpenAI Service
- AI Builder in Power Platform — how it connects to Azure AI
- Copilot Studio — understanding the AI layer underneath
- Practice scenarios and study notes for each exam skill area

## How this connects to the rest of the series

AI-900 builds naturally on [certifications-az900](https://github.com/HeatherAmi/certifications-az900). The Azure AI services also feed directly into the integration patterns covered in [certifications-pl400](https://github.com/HeatherAmi/certifications-pl400).
